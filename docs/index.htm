Background

A collaborative hub
for publishing data
from project cooperations.

ARCHAEOLOGY.LINK presents research data, ontologies, related research tools and web services for e.g., Linked Open Data technologies within the scope of the archaeological basic research carried out by the LEIZA and its cooperation partners in joint projects. This platform complements the LEIZA Archaeological Data Processing Web Service (ADP).

Components

ARCHAEOLOGY.LINK comprises LOD Research Data and LOD Tools.

  • Linked Open Samian Ware
  • Linked Open ARS
  • Linked Open Ships
  • Linked Archaeological Data Ontology
  • Wikidata Project archaeology.link
  • Wikidata Project Samian Ware
  • Wikidata Project ARS3D
  • NAVIS.one Maritime Thesaurus
  • Alligator
  • Academic Meta Tool

The LEIZA publishes resources on Wikidata and sets bidirectional links. Data is also linked to other resources, e.g. Pleiades Places.
